# Todo List

Uma aplicação de **lista de tarefas (ToDo List)** construída com **frontend e backend**, permitindo que você gerencie suas tarefas (criar, listar, atualizar e excluir). Este projeto foi feito para estudos.

---

## Tecnologias Utilizadas

### Backend
- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- Banco de dados postgres
- Lombok
- Validation
- Maven

### Frontend
- React
- JavaScript  
- Tailwind CSS

### Infra
- Docker & Docker Compose

---

## Funcionalidades

✔️ Criar novas tarefas  
✔️ Listar todas as tarefas  
✔️ Marcar tarefas como concluídas  
✔️ Editar tarefas existentes  
✔️ Excluir tarefas  
✔️ Execução com Docker para subir toda a aplicação com um comando

---

## Como Executar

Siga estas instruções para obter uma cópia do projeto funcionando em sua máquina local.

1. **Clone o repositório:**  
```sh
git clone https://github.com/AntonioNeto18/todo-list.git
cd todo-list
```

2. **Execute o projeto com docker-compose**
```sh
docker-compose up -d
```

Acesse `http://localhost:80`
